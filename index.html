The Steel programming language

Join the discussion
Steel is an open source general-purpose programming language propose to be friendly and efficient as C that helps you write better coding and tooling, build from scratch that focused on better developer experience. 
main {
  mut ideas = ['web', 'network', 'gaming']
  ideas use s {
    print 'Steel for ${s} and more!'
  }
}
Why Steel?

The idea of Steel as a community contributions project, aims to provide programmers an unprecedented combination of efficiency and expressiveness to build complex software fast; and it's also an ambitious effort to strike a balance between readability syntax and achievable performance that inherited some of the good ideas from other languages.

Currently in pre-alpha stage.
